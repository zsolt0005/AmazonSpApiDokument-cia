Create a new product

Creates / updates a product in the amazon catalog.

The requirements property must be LISTING_PRODUCT_ONLY for creating the listing only and not a new product with it.

Steps

  • Call the search items to ensure the product does not exist already.

  • Get the required attributes for creating a new product

    • Each product type can have different required properties that can be retrieved here

    • As the requirements pass LISTING_PRODUCT_ONLY to get the required parameter for the product creation

  • Call the listings API endpoint to create a new item in the amazon catalog (example below)

    • The /putListingsItem endpoint supports image submissions (1 main image + 9 secondary images)

    It's enough to put the image URLs in the body, and it should be downloaded and processed by the amazon API
