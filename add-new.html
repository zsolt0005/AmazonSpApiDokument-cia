Add new listing / Full update if exists

Creates / updates a listing for an existing product in the amazon catalog.

The requirements property must be LISTING_OFFER_ONLY for creating the listing only and not a new product with it.

Steps

  • Call the search items to ensure the product exist.

  • Get the required attributes for creating a new listing

    • Each product type can have different required properties that can be retrieved here

    • As the requirements pass LISTING_OFFER_ONLY to get the required parameter for the listing creation

  • Call the listings API endpoint to create a new listing (example below)

PUT https://sellingpartnerapi-eu.amazon.com/listings/2021-08-01/items/<seller id>/<SKU> ?marketplaceIds=<marketplace ids> x-amz-access-token: <access token> Content-Type: application/json { "productType": "SPORTING_GOODS", "requirements": "LISTING_OFFER_ONLY", "attributes": { "merchant_suggested_asin": [ { "marketplace_id": "A1PA6795UKMFR9", "type": "ASIN", "value": "B0CB92G9L3" } ], "externally_assigned_product_identifier": [ { "marketplace_id": "A1PA6795UKMFR9", "type": "upc", "value": "195871705458" } ], "condition_type": [ { "marketplace_id": "A1PA6795UKMFR9", "value": "new_new" } ], "fulfillment_availability": [ { "fulfillment_channel_code": "DEFAULT", "quantity": 2 } ], "purchasable_offer": [ { "marketplace_id": "A1PA6795UKMFR9", "currency": "EUR", "our_price": [ { "marketplace_id": "A1PA6795UKMFR9", "schedule": [ { "value_with_tax": 36 } ] } ] } ] } }

Seller ID is a publicly available ID number that identifies your storefront and products you sell on Amazon

SKU is a selling partner provided identifier for an Amazon listing.

Possible values for the Requirements can be found here

Attributes can be found as the output of the Get product type definition.
