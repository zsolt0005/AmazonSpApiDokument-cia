List orders

Get a list of orders by the given filter parameters.

GET https://sellingpartnerapi-eu.amazon.com/orders/v0/orders ?MarketplaceIds=<Marketplace Ids> &CreatedAfter=<Created After Date> &OrderStatuses=<Order statuses> x-amz-access-token: <access token>

Params

Name

Description

Required

CreatedAfter

A date used for selecting orders created after (or at) a specified time.

NO

CreatedBefore

A date used for selecting orders created before (or at) a specified time.

NO

OrderStatuses

A list of OrderStatus values used to filter the results.

NO

MarketplaceIds

A list of MarketplaceId values.

YES

FulfillmentChannels

A list that indicates how an order was fulfilled. Filters the results by fulfillment channel. Possible values: AFN (Fulfillment by Amazon); MFN (Fulfilled by the seller).

NO

NextToken

A string token returned in the response of your previous request.

NO

Either CreatedAfter or CreatedBefore must be specified!

Order Statuses

Status

Description

PendingAvailability

This status is available for pre-orders only. The order has been placed, payment has not been authorized, and the release date of the item is in the future.

Pending

The order has been placed but payment has not been authorized.

Unshipped

Payment has been authorized and the order is ready for shipment, but no items in the order have been shipped.

PartiallyShipped

One or more, but not all, items in the order have been shipped.

Shipped

All items in the order have been shipped.

InvoiceUnconfirmed

All items in the order have been shipped. The seller has not yet given confirmation to Amazon that the invoice has been shipped to the buyer.

Canceled

The order has been canceled.

Unfulfillable

The order cannot be fulfilled. This state applies only to Multi-Channel Fulfillment orders.
