Get a listing

Gets a listing if exists.

GET https://sellingpartnerapi-eu.amazon.com/listings/2021-08-01/items/<seller id>/<SKU> ?marketplaceIds=<marketplace ids> &IncludedData=<included data> x-amz-access-token: <access token>

Seller ID is a publicly available ID number that identifies your storefront and products you sell on Amazon

SKU is a selling partner provided identifier for an Amazon listing.

Included data

Value

Description

summaries

Summary details of the listing item.

offers

Current offers for the listing item.

fulfillmentAvailability

Fulfillment availability details for the listing item.
